# 🔥 GitChat

A sleek, real-time, room-based chat application built with Firebase that lets users instantly create or join chat rooms and start messaging — no signup required.

## 🚀 What Is GitChat?

GitChat is a lightweight browser-based chat app that allows users to:

- 👤 Enter a username

- 🏠 Create or join a chat room

- 💬 Send and receive messages in real time

- 🔐 Authenticate anonymously (no passwords required)

All messages are synced instantly using Firebase Firestore, making it perfect for quick collaboration, temporary chat rooms, or learning real-time web development.

- 🛠️ Tech Stack

- 🔥 Firebase App SDK

- 🗄️ Cloud Firestore – Real-time database

- 🔐 Firebase Authentication – Anonymous login

- 🌐 Vanilla HTML, CSS, JavaScript (ES Modules)

## ⚡ How It Works

### 1️⃣ Anonymous Login

When the page loads, Firebase automatically signs users in anonymously.

```signInAnonymously(auth)```

- No accounts. No passwords.

### 2️⃣ Join or Create a Room

Users enter:

- A username

- A room name

- If the room doesn’t exist — it’s created automatically.
